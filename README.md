# Proyecto DevPost 

Pequeno proyecto que simula un blog que tiene un cms donde se puede administrar el contenido del mismo. 

## Tabla de Contenidos

- [Instalacion](#Instalacion)
- [Uso](#uso)


## Instalacion

- PHP 7.4
- Apache 2.4.45
- Mysql 5.7.24
- Crear una BD llamada cms y volcar los datos del archivo cms.sql


## Tecnologias Usadas

- BULMA
- FONT AWESOME 5
- Jquery
